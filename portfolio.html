Project TARS

Status: In Progress
Tech Stack: Arduino, Python, Servo Motors


This is my flagship project: building a fully functional miniature version of TARS from the movie Interstellar. The goal is to replicate the unique walking mechanism using high-torque servos and integrate an AI voice assistant so it can actually converse like the character in the film. I am using my personal made assistant in python named "Nexus" which is using gemini api. Still i am working on the design of the model and because i am working on nexus v3 its still on hold, but i am sure that this will become true and practical very soon.


Currently working on the design and balancing algorithms.
